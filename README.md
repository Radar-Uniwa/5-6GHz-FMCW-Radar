# 📡 5-6GHz FMCW Radar 
This repository contains the design files for a 5–6 GHz FMCW radar developed as part of a diploma thesis at the University of West Attica. The repository contains the KiCad files for the Radar PCB, a MATLAB script that simulates the operation of a FMCW Radar and a CST file for the simulation of a coupled line coupler that is utilised on the Radar. 

This work is based on an original open source project by [Henrik Forstén](https://hforsten.com/third-version-of-homemade-6-ghz-fmcw-radar.html) and the original design files can be found [here](https://github.com/Ttl/fmcw3/tree/master). It is highly suggested that the original sources are studied first as this work follows the same workflow as the original while making a few hardware changes. 

## 🗂️ Folder Overview
The repository is structured as followed: 

-📁 Hardware: Contains the KiCad files for the Radar PCB.

-📁 MATLAB: Contains the MATLAB script developed that simulates the operation of a FMCW Radar, including Range-FFT, Doppler-FFT and Angle-FFT processing.

-📁 CST: Contains the CST simulation file for the coupled line coupler that is used on the Radar transmitter.
